# 数据准备指南

## 概述

本指南介绍如何准备和组织用于 ResNet 模型训练的 .npy 格式数据集。

## 数据集结构

数据集应按照以下目录结构组织：

```
Data/
├── 0/
│   ├── sample1.npy
│   ├── sample2.npy
│   └── ...
├── 1/
│   ├── sample1.npy
│   ├── sample2.npy
│   └── ...
├── 2/
│   └── ...
└── ...
```

### 目录说明

- **Data/**: 数据集根目录，可通过 `--data_dir` 参数指定
- **0/, 1/, 2/, ...**: 类别文件夹，每个文件夹代表一个类别
  - 文件夹名称将作为类别标签
  - 按字母顺序排序，确保标签映射一致
- **.npy 文件**: 每个样本的 2D 特征数据
  - 文件格式必须为 `.npy`
  - 建议数据形状：`(H, W)` = `(543, 512)`（单通道灰度图）

## 数据格式要求

### .npy 文件格式

1. **数据类型**: 任意浮点型（会自动转换为 float16）
2. **数据形状**: 2D 数组 `(H, W)`
3. **通道处理**: 加载时会自动添加通道维度 `(1, H, W)`
4. **数值范围**: 建议归一化到 [0, 1] 或 [-1, 1]

### 示例数据

```python
import numpy as np

# 创建示例数据
data = np.random.rand(543, 512).astype(np.float16)
np.save("Data/0/sample.npy", data)
```

## 数据集划分

数据集会自动按照以下比例划分：

- **训练集**: 60%
- **验证集**: 20%
- **测试集**: 20%

划分使用分层抽样，确保各数据集的类别比例一致。

### 修改划分比例

如需修改划分比例，编辑 `src/dataset.py` 中的 `data_set_split()` 函数参数：

```python
train_dataset, validate_dataset, test_dataset, labels__ = data_set_split(
    args.data_dir,
    train_ratio=0.6,   # 修改此处
    val_ratio=0.2,     # 修改此处
    test_ratio=0.2     # 修改此处
)
```

## 数据加载流程

1. 遍历 `Data/` 目录下的所有子文件夹
2. 读取每个子文件夹中的 `.npy` 文件
3. 建立标签映射：文件夹名 → 类别索引
4. 分层抽样划分训练/验证/测试集
5. 创建 `NPYDataset` 实例
6. 通过 `DataLoader` 批量加载

## 注意事项

1. **类别数量**: 默认支持 24 类，可通过 `--class_num` 参数修改
2. **文件一致性**: 确保所有 `.npy` 文件具有相同的形状
3. **数据质量**: 检查并清理损坏的 `.npy` 文件
4. **内存优化**: 数据以 float16 格式加载，节省 GPU 内存

## 故障排除

### 文件加载错误

如果遇到文件加载错误，检查：
- `.npy` 文件是否完整
- 文件路径是否正确
- 数据形状是否一致

### 类别不平衡

如果数据类别严重不平衡：
- 考虑使用加权损失函数
- 对少数类进行过采样
- 对多数类进行欠采样
